# We Are Going Racing

WAGR - We Are Going Racing. WAGR is an iRacing league for people who want to enjoy fun and
clean racing in an organized series. We're currently aiming to start one series
([F3](/series/f3/)), but would love to run many more!

Have ideas? Suggestions? Anything? Open a GitHub issue:
https://github.com/Robdel12/wearegoingracing/issues/new
Or join our Discord: https://discord.gg/GxueMZA
